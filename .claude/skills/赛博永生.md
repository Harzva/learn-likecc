# 赛博永生（个人知识库）

本仓库**不仅是** Claude Code 源码向课程与站点，也是**可 Git 版本化、可检索的个人知识库与赛博档案**。

## 何时参考本 Skill

- 新增或改写**个人随笔、项目经历、对外稿件**时，保持与现有栏目（`site/`、`wemedia/`）结构一致。
- 区分**技术讲义**（课程 S/D、手册）与**第一人称阅历**（专栏、知乎稿），避免混在同一文件里失真。
- 维护 **HTML ↔ `site/md/*.md` 双版本**与 `tools/check_site_md_parity.py` 约定。

## 核心约定

1. **起源**：learn-likecc / leaky CC — Source Map、可跑 CLI、架构讲义。
2. **扩展**：个人 Agent 工具链阅历、反代/工作流折腾（已脱敏）、后续**项目经历与作品集**入口将逐步挂到站点或 `wemedia/`。
3. **安全**：正文与脚本说明中**不出现**可复现的 API 密钥、端点、Cookie；私有路径仅作角色描述。
4. **永生**：依赖 **Git + 静态站 + Markdown** —— 换机器、换模型，仍能 diff、搜索、fork 自己的叙述；不是单一云文档锁定。

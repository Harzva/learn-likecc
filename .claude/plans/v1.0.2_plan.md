# v1.0.2_plan.md - 多窗口 Tab 模式起步版

## 目标

`v1.0.2-likecode` 先不追求完整分屏，而是把“同一个 session 下多个任务 tab”打成第一版可用能力。

这一版聚焦：

- tab 状态模型
- tab UI
- tab 切换与管理
- tab 级 transcript / model / provider 绑定
- subagent 面板占位
- `/show:slash`
- 启动红色爱心 logo

---

## 为什么是这一版

当前已经完成：

- 多 Provider 第一阶段
- `/model` 外部模型展示
- `/show` 配置查看
- 启动界面配置可见性
- 多窗口 / subagent 视图设计稿 V1

所以 `v1.0.2-likecode` 最合适的下一步，不是直接冲 pane，而是把 tab 模式落地，形成一个真正可交付、可演示、可继续扩展的版本。

同时，这一版也已经验证出一个重要判断：

- `tab` 和 `branch` 不重复
- 但当前轻量 tab 只是壳层
- 真正高价值的下一步应是 `panel / 分屏 / subagent 状态视图`

- 用户能看清每个 tab 的模型 / provider / 状态

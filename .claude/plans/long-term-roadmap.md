# learn-claude-code-source 长期规划

## 项目愿景

**目标**: 建立一个完整的 Claude Code 学习与恢复项目

五大主线:
1. **逆向工程** - 减少编译错误、恢复 Claude Code 功能
2. **课程体系** - 12-48 章节深度学习课程 + Python/Rust 版本
3. **自媒体运营** - 小红书、知乎、微信公众号文章发布
4. **文档完善** - 完整的逆向工程文档
5. **Awesome 汇总** - 收集所有泄露 claude-code 仓库

---

**方案 B**: 扩展到 48 章节

适合完整源码逐文件分析，每个重要模块独立成章。

在现有章节间插入更细致的内容:
- c02.1: Tool Discovery 机制

### 执行条件

**前提**: "将 c1-c12 迭代的好以后才有这个计划"

- c01-c12 全部完成 ✅

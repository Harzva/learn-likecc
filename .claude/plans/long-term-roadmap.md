# learn-claude-code-source 长期规划

## 项目愿景

**目标**: 建立一个完整的 Claude Code 学习与恢复项目

三大方向:
1. **课程体系** - 12-48 章节深度学习课程
2. **源码恢复** - 可运行的 Claude Code 版本
3. **文档完善** - 完整的逆向工程文档

---

## 一、课程扩展路线

### 当前版本 (v1.0.0)

**基础课程**: c01 - c12 ✅ 已完成

### 阶段 2: 扩展课程 (v1.1.0+)

用户建议: "当前的课程不是截止到c12，后面我们可以做更细致的课程"

**方案 A**: 扩展到 24 章节

| 章节范围 | 主题 |
|---------|------|
| c01-c12 | 基础架构 + 核心机制 |
| c13-c18 | 深度实现细节 |
| c19-c24 | 高级应用与实践 |

**方案 B**: 扩展到 48 章节

适合完整源码逐文件分析，每个重要模块独立成章。

**方案 C**: 中间章节

在现有章节间插入更细致的内容:
- c02.1: Tool Discovery 机制
- c02.2: Tool Input Validation
- c06.1: Subagent 通信协议
- c06.2: Fork 隔离实现
- c11.1: Vim Mode 状态机
- c11.2: Vim Keybinding 扩展

### 执行条件

**前提**: "将 c1-c12 迭代的好以后才有这个计划"

- c01-c12 全部完成 ✅
- 每章节有完整文档 + 代码示例 + 练习题 ✅
- 用户反馈积极
- 有足够源码分析深度支持扩展

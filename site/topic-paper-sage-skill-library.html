SAGE：Self-Improving Agent with Skill Library
自我进化技能库

这篇论文最值得看的，不是 RL 本身，而是 skill library 怎样从“静态能力包”变成 agent 持续进化的一部分。

论文与截图展示

本地 PDF / GitHub：打开 paper.pdf

核心解读

第一：SAGE 说明 skill library 不是只能做人写的外部包，也能进入 agent 的自我改进闭环。

第二：Sequential Rollout 这类设计，把“上一轮学到的技能能否帮下一轮”变成可训练、可累积的问题。

第三：这条线和我们站内的 `codex-loop in sleep` 很相关，因为它都在问：经验如何沉淀成未来可复用的能力。

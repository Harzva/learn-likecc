子专题 AI智能体

AI 智能体不是一个名词篮子
它们分属完全不同的壳层与场景

这页把 通用型 Agent、桌面工作台、OpenClaw 生态、浏览器 Agent 和 垂直行业 Agent 铺成一面墙。
鼠标悬停时，同一路线的工具会一起凸显，帮助读者一眼看出谁是同一条演化分支。

Map

先把 AI智能体 分成 6 条观察线

通用型 Agent

Manus、MiniMax Agent、Operator、Fairies 这类，重点是自主执行与泛任务能力。

桌面工作台

WorkBuddy、QoderWork、DuMate、Loomy 这类更像 AI 原生桌面空间。

OpenClaw 生态

OpenClaw、ArkClaw、AutoClaw、QClaw、RedClaw 这一串值得单独看成一系。

浏览器 Agent

Operator、Opera Neon、Fellou、BrowserOS、Dia 更像浏览器壳层的再发明。

垂直行业 Agent

科研、电商、金融、设计、内容创作都开始长出各自的专用智能体。

Agent 平台 / 社区

扣子、讯飞星辰Agent、MyShell、SkillHub 更像搭建和分发入口。

怎么逛这面墙更有效

想看通用自主执行：先看 Manus、MiniMax Agent、Operator、Genspark。

想看桌面工作台路线：看 QoderWork、WorkBuddy、DuMate，以及 OpenClaw 这一整系。

想看浏览器壳层怎么变成 Agent：看 Operator、Opera Neon、Fellou、BrowserOS、Dia。

想看行业型 Agent：按科研、电商、设计三条垂直线去看 SciMaster、遨虾、Lovart 等代表节点。

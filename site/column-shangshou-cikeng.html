上手与踩坑 赛博档案 · 实战篇 · Agent 实战手记

主标题：上手与踩坑。介绍语：本专栏同时作为 赛博档案 · 实战篇 与 Agent 实战手记——偏「能照着做 / 能避坑」的结论，与姊妹篇《从编辑器到 Agent》的阅历叙事互补。

正文将随项目经历逐步增量；当前页为专栏扉页。协作审阅可用 GitHub 上的 Markdown 同源文件。

专栏定位

仓库在源码学习之外，已演进为个人知识库与赛博档案（见 README 与 .claude/skills/赛博永生.md）。本专栏专门收：

  • 上手步骤：环境、命令、最小可复现路径（脱敏）；
  • 踩坑记录：现象 → 原因假说 → 解决办法 → 是否仍有效；
  • Agent 实战：Claude Code / MCP / Skills / Loop 等在真实任务里的用法与边界。

与课程 S/D 讲义的差别：讲义对齐架构与原理；本专栏对齐我当时怎么把它跑通。

与姊妹篇的关系

从编辑器到 Agent：一条工具链上的个人阅历讲「为什么摸、心路历程、工具变迁」。本专栏讲「这一步怎么做、别踩哪几个坑」。两篇都会持续长，但口气和用途分开，读者各取所需。

计划收录（占位，随写随钩）

  • 项目经历与作品集入口（多仓库 / 多页面时在此汇总链接）；
  • 分发与自动化（知乎 / 小红书 / Puppeteer 备忘等）的可操作结论；

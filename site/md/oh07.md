## 一句话

外部 MCP server 的 tools/resources 必须变成模型可见的**同质 Tool**；`McpToolAdapter` 用 `mcp__{server}__{tool}` 命名，并用 `create_model` 从 JSON Schema 动态建 Pydantic 入参模型。

## 关键路径

- `mcp/`：连接、`call_tool`。  
- `tools/mcp_tool.py`：`McpToolAdapter`、sanitize 与动态 `input_model`。  
- 资源类工具：list / read resource。

## 源码导读

- `_input_model_from_schema`：字段常宽松为 `object | None`，权衡「任意 MCP schema」与校验强度。  
- `create_default_tool_registry(mcp_manager)` 在 live manager 上遍历注册；断连或改配置后常需重启或 refresh（见 [OH04](oh04.md) `CommandResult`）。

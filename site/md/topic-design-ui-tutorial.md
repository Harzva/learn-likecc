## 概要

这页先回答一个实际问题：

- 什么场景该选 `LivePPT`
- 什么场景该选 `Slidev`
- 什么场景该直接上 `Remotion`

核心判断：

- deck / 演示页 → `LivePPT`
- developer slides / 讲义 → `Slidev`
- 程序化视频 / 动态叙事 → `Remotion`

## 各节摘要（对照 HTML）

### 子专题入口

用并列入口把三条纵深线分开：**论文**偏研究与方法、**热点**偏工程与社区讨论、**生态对比**偏产品形态与技能包设计，避免读者在单页里混杂预期。

#### 📄 论文分析

跳转到 `topic-agent-papers.html`，用于长期维护的精读与书目式笔记。

#### 🔥 技术热点

跳转到 `topic-agent-hot.html`，承载外文博客摘要表与抓取说明；现在也开始收官方项目主源样本，例如 `Hermes Agent` 这类 memory / skills / gateway 一体化项目，以及 `OfficeCLI` 这种面向 Agent 的 artifact-control CLI。

#### 🧭 生态对比

跳转到 `topic-agent-comparison.html`，聚焦 Superpowers/GStack、Subagents vs Teams 等对照叙事。

#### 🧠 Memory 机制

跳转到 `topic-memory-harness.html`：长期记忆的写入、整合、检索与安全（Mermaid 流程图 + 知乎稿 13 同源）。

#### 📄 VibePaper

跳转到 `topic-vibepaper.html`：把自动科研与自动论文系统单列成一条专题轴，当前最适合先读的是 `Autoresearch / ARIS` 与 `DeepScientist` 两条不同壳层。

#### 🧩 Superpowers vs Autoresearch

跳转到 `topic-superpowers-autoresearch.html`：基于 `reference/reference_agent/` 内 vend 的 `superpowers-marketplace` 与 `autoresearch / aris` 这条样本线做结构对照（知乎稿 21 同源）。

### Awesome · Agent 资源清单

聚合社区 Awesome 列表或等价精选链接，强调**第三方维护**与本站摘要的边界，方便一键扩展阅读。

### 相关课程章节

把专题与主线讲义（如 Agent Loop、MCP、Hooks、Subagent）做表格或列表映射，回答「看完专题该回哪几讲」；其中自动科研相关内容可继续跳到 `topic-vibepaper.html`。

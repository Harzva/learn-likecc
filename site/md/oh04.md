# OH04: CLI / 命令接口（对标 S04）

> **在线**: `/oh04.html` · **对照** [S04](s04.md)  
> **上游**: `cli.py`（Typer `app`、子命令 `mcp`/`plugin`/`auth`/`provider`/`cron`）、`__main__.py`（`python -m openharness`）、`commands/registry.py`（斜杠命令 `CommandRegistry`、`SlashCommand`、`CommandContext`）

## 要点

- **进程入口**：`__main__` 仅 `from openharness.cli import app` 后 `app()`。  
- **Typer**：默认无子命令时走交互会话；`-p`/`--print` 等非交互与课内 `claude -p` 叙事对应。  
- **斜杠命令**：大文件 `registry.py` 注册 `/compact`、`/model` 等，处理结果 `CommandResult`（`continue_pending`、`refresh_runtime` 等）驱动 TUI。

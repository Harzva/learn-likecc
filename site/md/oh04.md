## 一句话

用户从 shell 进入 Harness 时，需要**进程级 Typer 入口**与**会话内 `/command` 斜杠命令**两层命令面；`CommandResult` 可驱动 TUI 清屏、退出或 `continue_pending` 续跑，与 [OH01](oh01.md) Loop 外环衔接。

## 关键路径（读什么）

- `__main__.py`：仅 `from openharness.cli import app` 后 `app()`，统一包入口与 `python -m`。  
- `cli.py`：`typer.Typer` 根 app，子命令 `mcp` / `plugin` / `auth` / `provider` / `cron` 等；`invoke_without_command=True` 时可直接 `oh` 进交互。  
- `commands/registry.py`：斜杠命令注册与 `CommandResult`（`continue_pending`、`refresh_runtime` 等）。

## 源码导读

- **入口三行**：`__main__` 零逻辑委托，便于测试与打包。  
- **Typer**：`-p` / `--print` 等与课内管道式非交互叙事同题；用 `@app.callback` / `def main` 定位默认分支。  
- **Slash → Loop**：处理器异步返回 `CommandResult`，可触发不追加用户句而调用 `QueryEngine.continue_pending`。

## 与课内叙事对照

### 🎯 问题

Agent Loop 不断追加消息，会撞上模型上下文上限、抬高费用，并可能导致尾部截断丢失关键指令或工具结果。

### 💡 解决方案

Claude Code 采用分层策略：预警、自动压缩、错误阈值下的强制压缩等，在体验与可恢复性之间折中。在线页用树状 ASCII 说明分级。

### 📐 阈值计算

结合 `getContextWindowForModel`、预留输出与安全 buffer，解释自动压缩触发线如何随模型变化。实现片段见 `autoCompact.ts` 引用（在线页）。

### 🔑 压缩类型

区分 Auto Compact、手动 `/compact`、Micro Compact、Memory Compact 等路径的触发主体与粒度。卡片式说明见在线页。

### 📊 压缩流程

概述从 pre-compact hooks 到真正改写 messages 的步骤化流程；完整 `compact()` 逻辑与注释见在线页。

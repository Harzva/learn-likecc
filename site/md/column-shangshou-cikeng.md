## 专栏定位

**主标题：上手与踩坑。** 本专栏同时作为 **赛博档案 · 实战篇** 与 **Agent 实战手记**——偏「能照着做 / 能避坑」的结论，与姊妹篇 [《从编辑器到 Agent》](https://harzva.github.io/learn-likecc/column-agent-journey.html) 的**阅历叙事**互补。

仓库在源码学习之外，已演进为**个人知识库与赛博档案**（见根目录 README 与 `.claude/skills/赛博永生.md`）。本专栏专门收：

- **上手步骤**：环境、命令、最小可复现路径（脱敏）；
- **踩坑记录**：现象 → 原因假说 → 解决办法 → 是否仍有效；
- **Agent 实战**：Claude Code / MCP / Skills / Loop 等在真实任务里的用法与边界。

与课程 S/D 讲义的差别：讲义对齐**架构与原理**；本专栏对齐**我当时怎么把它跑通**。

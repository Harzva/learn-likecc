# AI杂货铺

AI杂货铺不是一个“工具站镜像页”，而是一间按场景分货的专题入口。

当前先搭五个货架：

1. CLI Agent
2. AI 编辑器
3. Agent 开发工具
4. 模型 API / 聚合
5. 模型评测

## 选货原则

- 不按导航站原分类照搬，而按真实使用场景分货。
- 先收会反复打开的入口，而不是追求一口气收满。
- 每个货架都要说明“为什么值得收”。
- 与站内 `Agent`、`工具链`、`大模型` 等专题互补。

## 各节摘要（对照 HTML）

### 🚀 快速开始

把最小闭环（安装→启动→第一条指令→短教程）压缩在一节，降低与 `tutorial.html` 的重复感的同时保留**独立可读性**。

#### 安装 Claude Code / 启动对话 / 第一个命令 / 15 分钟上手

子步骤分别对应环境准备、交互入口、验证成功与稍长一点的引导阅读，形成递进。

### 🗺️ 学习路线图

用阶段化路径（基础→工作流→扩展→源码）帮助读者把本站教程、讲义与专题**排进个人计划**。

### 📖 功能详解

对七大能力模块各给一段「是什么、何时用、与何者易混」的说明，并链向教程或 S 课中更细的实现向内容。

#### Slash Commands / Memory / Skills / Hooks / MCP / Subagents / Plugins

每块强调用户可见行为与配置入口，而非实现细节；Hooks/MCP/Subagents 与课程章节名称对齐，便于交叉检索。

### ❓ 常见问题

汇总典型排错与概念误区（权限、上下文、模型切换等），以**问答体**节省翻论坛的时间。

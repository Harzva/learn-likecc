# 模型评测专页

这页重点讲清楚四个入口分别该怎么看：

- MMLU
- MMBench
- OpenCompass
- HELM

## 先记住

- MMLU：经典文字知识 benchmark
- MMBench：多模态 benchmark
- OpenCompass：评测平台 / 榜单聚合入口
- HELM：透明评测方法和可复现框架

## 最实用的顺序

1. 先区分文字能力还是多模态能力
2. 再看 OpenCompass 的横向宽表
3. 最后用 HELM 补方法论和透明度

## 评测地图

这页还补了一整批评测入口的语境化分拣：

- Arena / 对战平台：MagicArena、LMArena
- 文字 benchmark：MMLU、C-Eval、CMMLU、AGI-Eval、LLMEval3
- 多模态 benchmark：MMBench
- 平台 / 榜单：Open LLM Leaderboard、FlagEval、OpenCompass、H2O EvalGPT
- 方法论 / 框架：HELM
- 垂直数据集：PubMedQA

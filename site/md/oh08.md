## 一句话

长耗时 Shell 或子 Agent 不应阻塞主 turn：`BackgroundTaskManager` 登记任务，通过 `task_output` / `task_list` 等工具拉取与停止；主 `run_query` 仍按轮推进。

## 关键路径

- `tasks/manager.py`：`BackgroundTaskManager`、`get_task_manager`。  
- `tasks/types.py`：`TaskRecord`、状态枚举。  
- `tools/task_*_tool.py`：create、list、get、stop、output、update。

## 源码导读

- 任务在 asyncio 或子进程后台跑；主循环用 tool 拉输出，故需要稳定 **task id**。  
- `ui/backend_host.py` 引用 task manager 做展示——任务系统不限于纯 TUI。

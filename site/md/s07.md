## 概要

本讲说明 **MCP（Model Context Protocol）**：与外部工具/数据源对接的基本概念、配置要点及在 Claude Code 生态中的典型用法。

### 🎯 问题

内置工具无法覆盖数据库、内部 API、私有文档等；需要标准协议把外部能力暴露为模型可调用的资源与工具。

### 💡 解决方案

通过 MCP 在宿主与外部 Server 之间建立工具/资源目录，生命周期内复用连接，减少重复握手成本。

### 🔑 传输类型

对比 stdio、HTTP(S) 等传输的部署特征与安全面，帮助选型。细节表与图示见在线页。

Local Event Stream

本地 SSE 联调

当你想把仿真和真实事件流接起来，可以在本机跑 relay，再让浏览器订阅事件。

  1. 1终端一（仓库根）：python3 tools/cc_loop_sse_relay.py（可加 --fast）
  2. 2终端二：cd site && python3 -m http.server 8080
  3. 3浏览器打开 http://127.0.0.1:8080/topic-cc-loop-lab.html
  4. 4确认 URL 为 http://127.0.0.1:8769/events，点连接

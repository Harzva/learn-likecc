Loop 事件流 本地实验室

本页用浏览器 EventSource 订阅本机 SSE 演示流，用于理解 projects/cc-loop-live 中的事件模型。非 GitHub Pages 线上能力：须在你电脑上同时跑静态站 + 中继。

步骤

  1. 终端一（仓库根）：python3 tools/cc_loop_sse_relay.py（可加 --fast）
  2. 终端二：cd site && python3 -m http.server 8080
  3. 浏览器打开 http://127.0.0.1:8080/topic-cc-loop-lab.html（勿用 file://）
  4. 确认下方 URL 为 http://127.0.0.1:8769/events（与脚本默认端口一致），点连接

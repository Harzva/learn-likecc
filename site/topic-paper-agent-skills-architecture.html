核心解读

第一：这篇论文把 skills 定义成一种高于 tool、低于 agent runtime 的能力抽象层。

第二：它强调 progressive disclosure、portable skill definitions 和 MCP 的互补关系，这和我们站内一直在讲的 `skill + runtime + context` 三角非常对齐。

第三：它不只讲架构，还把 acquisition、deployment、security 一起收进来，所以适合作为这条专题的总引子。

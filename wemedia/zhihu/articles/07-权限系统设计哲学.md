## 三层权限架构

Claude Code 采用了三层权限控制：

```
┌─────────────────────────────────────────┐
│           用户授权层                     │
│  (弹窗确认、记住选择、信任项目)          │
├─────────────────────────────────────────┤
│           沙箱隔离层                     │
│  (文件系统隔离、网络限制、命令过滤)      │
├─────────────────────────────────────────┤
│           系统防护层                     │
│  (敏感路径保护、危险命令拦截)            │
└─────────────────────────────────────────┘
```

### 第一层：用户授权

```typescript
// 权限请求流程
async function requestPermission(action: Action): Promise<boolean> {
  // 1. 检查是否已信任
  if (isTrusted(action)) {
    return true
  }

  // 2. 检查是否有记住的选择
  const saved = getSavedPermission(action)
  if (saved !== undefined) {
    return saved
  }

  // 3. 弹出确认框
  const result = await showPermissionDialog({
    action: action.type,
    details: action.details,
    options: ['允许', '拒绝', '始终允许'],
  })

  // 4. 保存选择
  if (result === 'always_allow') {
    savePermission(action, true)
  }

  return result !== 'deny'
}
```

## 实战：绕过权限的风险

源码中记录了用户尝试绕过权限的案例：

### 案例一：符号链接攻击

```bash
# 用户尝试通过符号链接访问受保护文件
ln -s ~/.ssh/id_rsa ./allowed_dir/key
```

**Claude Code 的防护**：

```typescript
function resolvePath(path: string): string {
  const resolved = fs.realpathSync(path)
  if (isProtectedPath(resolved)) {
    throw new PermissionDenied(resolved)
  }
  return resolved
}
```

### 案例二：命令注入

```bash
# 用户尝试通过命令注入执行危险操作
cat file.txt; rm -rf /
```

**Claude Code 的防护**：

```typescript
function validateCommand(cmd: string): boolean {
  const parts = parseCommand(cmd)
  for (const part of parts) {
    if (DANGEROUS_COMMANDS.includes(part)) {
      return false
    }
  }
  return true
}
```

## 最佳实践

### 对于用户

1. **审查权限请求**：不要习惯性点击"允许"
2. **使用信任项目**：只在信任的项目中启用自动权限
3. **定期清理**：检查并清理已保存的权限设置

### 对于开发者

```typescript
// 好的实践：声明式权限请求
async function readFile(path: string) {
  // 明确声明需要什么权限
  await requestPermission({
    type: 'fs.read',
    resource: path,
    reason: '读取文件内容用于分析',
  })

  return fs.readFile(path)
}

// 不好的实践：过度请求
async function readFile(path: string) {
  // 不要请求不需要的权限
  await requestPermission({
    type: 'fs.*',  // 太宽泛
    resource: '/', // 整个文件系统
  })
}
```

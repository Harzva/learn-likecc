## 最受关注的 5 大隐藏功能

### 1. KAIROS - 永远在线的 AI 代理

```typescript
// 源码中的 KAIROS 配置
const KAIROS_CONFIG = {
  enabled: feature('KAIROS'),
  daemon: true,           // 后台守护进程
  webhook: true,          // 支持 webhook 订阅
  githubPR: true,         // GitHub PR 监控
  scheduled: true,        // 定时任务

```

**功能描述**：
- 让 Claude Code 成为 24/7 在线的 AI 代理
- 即使关闭终端也能持续运行
- 支持 webhook 订阅、GitHub PR 监控、定时任务

**应用场景**：
- 夜间自动处理 PR review
- 监控服务器状态并自动修复
- 定时执行重复性任务

**状态**：高度实验性，目前无公开启用方式

### 3. Coordinator Mode - 多代理编排系统

```typescript
// Coordinator 架构
const COORDINATOR = {
  main: 'coordinator',     // 主 Claude
  swarm: ['research', 'synthesis', 'implementation'],
```

**功能描述**：
- 一个主 Claude 协调多个子 Agent
- 分为研究-合成-实现等结构化阶段
- 支持临时工作区和隔离 Git worktrees

**工作流程**：
```
用户请求
    │
    ▼
┌─────────────┐
│ Coordinator │ ← 主 Claude
└─────────────┘
    │
    ├──▶ Research Agent   → 收集信息
    ├──▶ Planning Agent   → 制定方案
    ├──▶ Code Agent       → 编写代码
    └──▶ Review Agent     → 审核质量
```

**核心价值**：将复杂任务分解、并行处理

### 4. AutoDream - 梦境记忆整合系统

```typescript
const AUTO_DREAM = {
  trigger: 'idle | night',    // 触发条件
  operations: [
    'merge_observations',     // 合并观察
    'eliminate_contradictions', // 消除矛盾
    'distill_memories',       // 记忆蒸馏
    'self_maintenance',       // 自我维护
  ],
  goal: 'prevent_memory_bloat',  // 防止记忆膨胀
}
```

**功能描述**：
- 在用户空闲或夜间自动运行
- 合并观察、消除矛盾、将模糊笔记转为事实
- 支持记忆蒸馏和自我维护

**仿生学原理**：
> 人类睡眠时会整理白天的记忆，消除矛盾，强化重要信息。AutoDream 就是 AI 版的"做梦"。

**解决的问题**：
- 长期会话的记忆膨胀
- 矛盾信息的自动修正
- 重要信息的长期保留

### 5. Undercover Mode - 员工专属隐藏模式

```typescript
const UNDERCOVER = {
  enabled: isAnthropicEmployee(),
  features: {
    hideAIIdentity: true,     // 隐藏 AI 身份
    removeCoAuthoredBy: true, // 去除贡献记录
    customPrompt: true,       // 自定义系统 prompt
  },
  noOptOut: true,             // 无关闭开关
}
```

**功能描述**：
- 专为 Anthropic 员工设计
- 注入系统 prompt 让 Claude 隐藏 AI 身份
- 不提及自己是 AI
- 去除贡献记录中的 Co-Authored-By 等痕迹

**争议点**：
- 员工专属，无强制关闭开关
- 可用于隐蔽的 AI 辅助工作
- 伦理问题：用户有权知道是否在与 AI 交互？

## 如何发现这些功能？

源码中的 feature flag 机制：

```typescript
// 源码中的 feature flag 定义
const FEATURES = {
  KAIROS: 'kairos_enabled',
  ULTRAPLAN: 'ultraplan_enabled',
  COORDINATOR: 'coordinator_mode',
  AUTO_DREAM: 'auto_dream_enabled',
  UNDERCOVER: 'undercover_mode',
  // ... 还有 39 个
}

function feature(name: string): boolean {
  return MACRO[FEATURES[name]] ?? false
}
```

通过分析 `feature()` 函数的调用，我们发现了所有隐藏功能。

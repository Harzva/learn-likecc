## 数字指纹：你的身份早已暴露

泄露的源码显示，Anthropic 的用户追踪系统极其复杂，通过多层指纹识别锁定用户：

### 第一层：网络环境

| 指标 | 说明 |
|------|------|
| IP 地址 | VPN 也能被识别，因为要综合判断 |
| 时区 | UTC+8 直接暴露地理位置 |
| 系统语言 | zh-CN 是高风险标签 |

### 第二层：设备信息

- 操作系统版本
- 浏览器指纹
- 屏幕分辨率
- 安装的字体列表

### 第三层：开发环境

最离谱的是，源码显示 Anthropic 会读取你的 `.git` 配置：

```javascript
// 源码中的遥测采集
```

这意味着什么？你的 Git 提交记录、GitHub 账号，全部被关联了。

## 防封指南

基于源码分析，这里是最有效的防封策略：

### 1. 环境隔离

```bash
# 使用独立的浏览器配置
chromium --user-data-dir=/path/to/isolated/profile
```

### 2. 干净的 IP

- 避免使用公共 VPN
- 选择住宅 IP 而非机房 IP
- 定期更换 IP 但保持地区一致

### 3. 指纹浏览器

使用 Multilogin、GoLogin 等工具，创建独立的浏览器指纹。

### 4. 初期策略

- 前 7 天避免使用中文对话
- 不要使用官方桌面客户端（链条越长暴露越多）
- 避免第三方代理工具

### 5. Git 配置清理

```bash
# 使用临时的 git 配置
git config --local user.name "anonymous"
git config --local user.email "anonymous@example.com"
```
